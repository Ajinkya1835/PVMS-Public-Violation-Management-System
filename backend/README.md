# PVMS Backend API

Node.js + Express REST API for the Public Violation Management System.

## Quick start

```bash
cp .env.example .env
npm install
npm run dev
```

API runs at **http://localhost:5000**

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start with nodemon |
| `npm run start` | Production start |
| `npm run seed:demo` | Seed test users and demo data |
| `npm run test:api` | API smoke tests |
