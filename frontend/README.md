# PVMS Frontend

React 19 + Vite portal for citizens, permit holders, and officers.

## Quick start

```bash
cp .env.example .env
npm install
npm run dev
```

Portal runs at **http://localhost:5173**

## Development

| Command | Description |
|---------|-------------|
| `npm run dev` | Vite dev server |
| `npm run build` | Production build → `dist/` |
| `npm run lint` | ESLint |

## Test page (dev only)

http://localhost:5173/test-info
